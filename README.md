# Magento 2 Extension for Ajax add to cart
It will show modal when user clicks on the add to cart

<p>Features<p>
  <h3>Easy to use</h3>
<p>This extension is easy to use.</p>

<h3>No Page load</h3>
<p>It will fire ajax request on the click event of the add to cart.</p>
